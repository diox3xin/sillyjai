# Card Tools for SillyTavern

Кроссплатформенное расширение для SillyTavern: работает на ПК, iOS и Android, потому что не обращается к внешним сайтам и не использует popup-окна.

## Возможности

### 🌐 Translate keys

Переводит ключевые слова лорбука (`key` и `keysecondary`) на выбранный язык.

- Можно выбрать текущий привязанный World Info или конкретный лорбук из списка.
- Перевод идёт через LLM, уже подключённую в SillyTavern.
- Дубликаты после перевода удаляются.
- Для русского промпт просит именительный падеж и единственное число.

### ✏️ Rename in card

Заменяет имя/фамилию/термин в текущей открытой карточке персонажа.

- Можно добавить несколько пар `Find -> Replace`.
- Есть режим `Whole words only`.
- Есть режим `Case sensitive`.
- Сначала показывается предпросмотр: сколько замен будет сделано в каждом поле.
- Только после предпросмотра становится доступна кнопка `Apply`.

Поля карточки, которые обрабатываются:

- `name`
- `description`
- `personality`
- `scenario`
- `first_mes`
- `mes_example`
- `alternate_greetings`
- `creator_notes`
- `creatorcomment`
- `system_prompt`
- `post_history_instructions`

## Установка

1. Открой SillyTavern.
2. Extensions -> Install Extension.
3. Укажи репозиторий или установи папку расширения.
4. После перезагрузки в панели Extensions появится блок **Card Tools**.

## Важно

Это расширение больше не импортирует персонажей/лорбуки из JanitorAI. Старый функционал был удалён, потому что автоматический доступ к чужому сайту из обычного ST-расширения нестабилен и не кроссплатформенен.
